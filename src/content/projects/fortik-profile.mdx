## Overview

Fortik adalah website company profile yang dikembangkan untuk perusahaan konstruksi PT. Fortik Indonesia. Website ini menampilkan portofolio proyek, layanan, dan informasi perusahaan dengan desain yang modern dan profesional.

## Key Features

### 1. Project Management System
- Dashboard untuk mengelola proyek konstruksi
- Upload dokumentasi dan progress foto
- Timeline tracking untuk setiap proyek
- Client portal untuk monitoring

### 2. Dynamic Content
- Content Management System (CMS) untuk update konten
- Blog system untuk artikel dan berita perusahaan
- Gallery untuk showcase project photos
- Testimonial section dari klien

### 3. Performance Optimization
- Server-side rendering dengan Next.js
- Image optimization otomatis
- Lazy loading untuk performa optimal
- SEO-optimized untuk ranking Google

## Technical Stack

Website ini dibangun menggunakan teknologi modern:

- **Frontend**: livewire
- **Styling**: Tailwind CSS untuk design system yang konsisten
- **Backend**: Laravel untuk authentication dan database
- **Database**: MySQL
